Getting Started

Python Installation

There are many ways to install python. The recommended method here is to install Miniconda, which can be found here.

Note: Anaconda includes everything in Miniconda, and so can be used instead if it is installed. Also, a new miniconda installation will not interfere with an existing ordinary python installation.

Once Miniconda is installed, open up terminal and create an empty environment.

conda create -n my_environment
conda activate my_environment

Install the version of python supported by your version of Planetary:

PlanetaryPython
v1.14v2.7.3
v2.4.3v2.7.3
v3.2v3.6.8
v4.0.1v3.7.7

Notice in particular that the latest version of planetary does not necesarily play well with the latest python version.

conda install python=3.7.7

Install Planetary with pip:

pip install planetary2d

If all went well, the following command should launch a test app:

python -m planetary2d.examples.ninja

And that's it! To learn about how to use the engine, see the wiki. The issues page is a good place to check if something is going wrong. Finally, join the community on Discord!
